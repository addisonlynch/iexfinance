Welcome to iexfinance’s documentation!¶

iexfinance’s documentation is organized into the following sections:

  • Getting Started - Installation instructions, basic usage information, migrating to IEX Cloud

  • Endpoints - 1:1 mirror of the IEX Cloud documentation

  • Package Information - additional package information, developer/testing documentation

Warning

IEX will end support for the Legacy v1 Devleoper API on June 1st, 2019. At that time, the use of iexfinance will require an IEX Cloud account. In addition, support for all legacy only endpoints will be discontinued.
